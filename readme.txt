Description:
The website gets colors and related information from an external API - ('https://reqres.in/api/products'). Colors and related information are shown in the table. Five colors are shown per page. You can click on any color and get detailed information. You can also choose the color by ID by typing the id of color into the input field at the top of the page.

Technologies:
Stack: React, Redux Toolkit, TypeScript, Axios, Material UI, SASS.

Quick Setup:
Download the repo by using in your terminal git clone https://github.com/Murad-Git/miniature-chainsaw then go to the main folder and start the project by:
cd miniature-chainsaw

Install all dependencies:
npm i
or
yarn install

Run in development mode:
npm run start
or
yarn start

To run in production mode:
npm run build
or
yarn build
